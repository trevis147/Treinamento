# Treinamento
Criar um formulario de cadastro de usuario com os seguintes campos:
Nome
Email
Senha
Confirmar Senha
Cep
Rua
Bairro
Cidade
Estado
Botao de submit

esse botao precisa pegar todas as informações do formulario criado e inicialmente mostrar no console (F12) do browser
